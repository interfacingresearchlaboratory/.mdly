MONOid is a planning system built around **Containers → Projects → Tasks**, with **Routine Blocks** and **Reviews** to help you shape a realistic week and build a sustainable cadence over time.

If you need help, want to report an issue, or want to support development, here are the best ways to reach us.

## Getting Help

If you have questions, need assistance, or want to report a bug:

- **Email**: support@usemonoid.com
- **Documentation**: Check the docs for core concepts, conventions, and workflows
- **Issue reporting**: If you’re part of a team rollout, share reproduction steps + screenshots when possible

> When reporting an issue, include: what you expected, what happened instead, and where you were in the app (Calendar / My Inbox / Kanban / Index pages).

## Support Options

### Product Support

**For individual users**, the best ways to get help:

- **Message us** — Reach out via the in-app feedback or support channel (best place for quick questions)
- **Join us on X** — Follow [@usemonoid](https://x.com/usemonoid) for updates and to ask questions
- **Docs** — Browse this documentation for concepts, conventions, and workflows

**For enterprise**, we offer additional support with:

- Integrating into your existing stack
- Migrating over to MONOid
- Setting up your Containers / Projects structure
- Designing Routine Blocks that match your actual week
- Establishing a review cadence (Daily / Weekly / Monthly / Quarterly / Annual)
- Team workflows and rollouts

For support, contact: **support@usemonoid.com**

## Downloads

MONOid is available as a desktop application and as a web app.

### Desktop Application

Download the MONOid desktop app for your operating system:

- **macOS**: DMG or ZIP
- **Windows**: Installer
- **Linux**: AppImage or DEB

The desktop app provides:

- Native window management
- Persistent authentication sessions
- Deep linking support
- Connectivity handling
- System integration

### Web Application

You can also use MONOid in your browser at **usemonoid.com**.

The web version provides the same core functionality and works on any modern device and browser.

### Installation

After downloading the appropriate installer for your platform:

1. Run the installer
2. Follow the installation prompts
3. Launch MONOid from your applications folder
4. Sign in or create an account to get started

The desktop app will automatically check for updates and notify you when new versions are available.
