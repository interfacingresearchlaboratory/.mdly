MONOid supports keyboard shortcuts for fast navigation and creation. Shortcuts are disabled when you're typing in an input field, text area, or rich text editor.

On macOS, <kbd>Cmd</kbd> is used as the modifier key. On Windows and Linux, use <kbd>Ctrl</kbd> instead.

## Create actions

| Shortcut | Action |
|---|---|
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>C</kbd> | Create Container |
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd> | Create Project |
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>T</kbd> | Create Task |
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>R</kbd> | Create Review |
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd> | Add Routine |
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>E</kbd> | Add Event (calendar) |
| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>M</kbd> | Create Template |

These work from anywhere in the app and open the corresponding creation dialog.

## Calendar navigation

| Shortcut | Action |
|---|---|
| <kbd>J</kbd> | Previous day (day view) or previous week (week view) |
| <kbd>K</kbd> | Next day (day view) or next week (week view) |
| <kbd>T</kbd> | Jump to today |
| <kbd>V</kbd> | Toggle between day and week view |
| <kbd>Cmd</kbd> + <kbd>←</kbd> | Toggle left sidebar (shaping/todo tasks) |
| <kbd>Cmd</kbd> + <kbd>→</kbd> | Toggle right sidebar (backlog tasks) |

## Views and filters

| Shortcut | Action |
|---|---|
| <kbd>Shift</kbd> + <kbd>V</kbd> | Open display config (grouping, sorting, visible properties) |
| <kbd>Shift</kbd> + <kbd>F</kbd> | Open filter dropdown |
| <kbd>X</kbd> | Toggle theme (light/dark) |

## Tips

- <kbd>Cmd</kbd> + <kbd>Enter</kbd> submits most dialogs (create, edit, save).
- Calendar navigation shortcuts (<kbd>J</kbd>, <kbd>K</kbd>, <kbd>T</kbd>, <kbd>V</kbd>) only work when the Calendar is focused and you're not typing in an input.
- Use <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>T</kbd> from any page to quickly capture a task without navigating away.
